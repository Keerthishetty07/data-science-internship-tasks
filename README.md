# 📊 Data Science Internship Tasks

## 📌 Project Overview

This repository contains the tasks completed during my **Data Science Internship** using Python. The project covers Easy, Medium, and Hard level tasks related to **data analysis, data cleaning, visualization, and statistical analysis**.

The main objective of this project is to understand the basics of **Data Science** and perform real-world data analysis using Python libraries.

The dataset used in this project is the **Student Performance Dataset**, which contains information about student exam scores and performance.

---

## 🎯 Objectives

The objectives of this internship project are:

* ✅ To learn how to load and explore datasets
* ✅ To perform statistical analysis on data
* ✅ To visualize data using charts and graphs
* ✅ To clean raw datasets and handle missing values
* ✅ To analyze relationships between variables
* ✅ To perform complete data analysis and generate insights
